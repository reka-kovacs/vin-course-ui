## VIN Course UI

This app reads participant course progress data from SQL server and displays it in a sortable table.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Participants as rows, courses as columns
- Table entries show completion and last accessed
- Color coded by completion %
- Sortable columns
- Search by Participant ID
- Pagination
